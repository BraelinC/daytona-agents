Quick Start

  1. Run python run_terminal.py to create sandbox
  2. Copy the Terminal URL into the field above
  3. Click "Open Terminal"
  4. In terminal, run: tmux attach -t main

TMUX Shortcuts

  • Ctrl+A 1/2/3 - Switch between windows
  • Ctrl+A | - Split pane vertically
  • Ctrl+A - - Split pane horizontally
  • Ctrl+A d - Detach (keep session running)
  • Ctrl+A c - Create new window
